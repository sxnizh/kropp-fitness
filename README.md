# Kropp Fitness

Небольшой учебный лендинг для фитнес-клуба Kropp Fitness.  
Проект сделан на чистом HTML и CSS без JavaScript и сборки.

## Демо

https://sxnizh.github.io/kropp-fitness/

## Технологии

- HTML
- CSS

## Что есть в проекте

- адаптивная верстка
- шапка с навигацией и кнопкой бронирования
- hero-секция с крупным заголовком
- блок с информацией о фитнес-событии
- секция Motivation с карточками
- секция Type of training
- адаптация под разные размеры экрана
- кастомные шрифты через `@font-face`
- hover-эффекты для кнопок и ссылок

## Как запустить

1. Клонировать репозиторий

```bash
git clone https://github.com/sxnizh/kropp-fitness.git
```

2. Открыть папку проекта

```bash
cd kropp-fitness
```

3. Открыть `index.html` в браузере
